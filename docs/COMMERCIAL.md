# Commercial Delivery

Chroma Beauty SDK is intended for commercial integration, private deployment, and customized image effect delivery.

## Deliverables

Depending on the cooperation model, delivery can include:

- Android SDK package
- iOS SDK package
- Demo app
- Sample project
- API documentation
- Model/resource package
- Scene presets
- Performance tuning suggestions
- License verification
- Version upgrade support

## Evaluation Scope

An evaluation package can be prepared around a small set of agreed scenarios:

- Selfie portrait
- Group portrait
- Stage/event image
- Low-light image
- E-commerce portrait or product-person image
- Professional profile photo

## Not Included in This Public Repository

- Production SDK binaries
- Model weights
- Training data
- Private source code
- Commercial license keys
- Customer-specific effect presets
