## The Problem Is Not a Lack of Filters

Most mobile photo products already have filters. The harder problem is making user photos look consistently good across real-world conditions.

Portrait images can be difficult:

- indoor lighting is mixed;
- LED screens can create strong color casts;
- group photos need consistent skin tones;
- global enhancement can make faces too red, yellow, gray, or saturated;
- beauty effects can become blurry or artificial;
- preview and export should remain consistent.

Chroma Beauty SDK is designed for this kind of mobile imaging workflow. It combines **AI color grading**, **portrait-safe enhancement**, **beauty retouching**, **face shaping**, and **preview/export consistency** into one integration-ready pipeline.

## Demo Workflow

![Demo Flow](https://raw.githubusercontent.com/18818474455/ChromaBeautySDK/main/assets/apk-demo/demo-flow.gif)

The demo app follows a practical editing path:

```text
Import image
-> Run AI color recommendation
-> Keep portrait protection enabled
-> Fine-tune color, style and beauty parameters
-> Compare before / after
-> Export the final result
```

This gives casual users an instant result while still giving advanced users control over the final image.

## AI Color Grading That Remains Editable

![AI One-Tap Color](https://raw.githubusercontent.com/18818474455/ChromaBeautySDK/main/assets/apk-demo/one-tap-color.jpg)

One-tap enhancement is useful only when the result can still be adjusted.

Chroma can expose AI recommendations as editable parameters:

| Area | Examples |
|---|---|
| Light | Exposure, contrast, highlights, shadows |
| Color | Temperature, tint, vibrance, saturation |
| Local color | HSL-style controls, color mixer direction |
| Style | LUT-style looks, curves, portrait styles |
| Protection | Skin tone, highlights, white clothing, portrait area |

For product teams, this means presets, AI recommendations, manual sliders, and export settings can share one coherent processing model.

## Beauty Retouching Without a Plastic Look

![Beauty Controls](https://raw.githubusercontent.com/18818474455/ChromaBeautySDK/main/assets/apk-demo/beauty-shape-controls.jpg)

Commercial beauty processing should not simply blur the whole image. Users expect cleaner skin, but they also expect eyes, hair, clothing, edges, and background details to remain visually clear.

Available product directions can include:

- skin smoothing;
- whitening and rosy tone;
- shine control;
- dark circle correction;
- face shaping;
- body reshape;
- beauty parameters that can be exposed in the app UI.
