# Article Assets

这个目录用于整理 Chroma Beauty SDK 的公开展示素材，适合放在 GitHub README、CSDN 文章、商务介绍页或 SDK 试用说明中。

## 推荐配图顺序

1. 首屏总览图：`assets/apk-demo/demo-storyboard.jpg`
   - 用途：文章开头或 GitHub 首屏。
   - 重点：一张图展示 Demo 工作台、一键调色、美颜美型三段流程。

2. 操作流程动图：`assets/apk-demo/demo-flow.gif`
   - 用途：功能介绍段落之后。
   - 重点：展示从打开 Demo 到切换参数面板的真实操作路径。

3. 一键调色截图：`assets/apk-demo/one-tap-color.jpg`
   - 用途：说明 AI 一键调色、人像保护、强度滑杆。
   - 重点：突出“AI 推荐后仍可人工微调”的产品体验。

4. 颜色与风格面板：`assets/apk-demo/color-panel.jpg`、`assets/apk-demo/style-panel.jpg`
   - 用途：说明 SDK 不只是滤镜，也提供可解释、可调节的参数体系。
   - 重点：颜色、风格、LUT、HSL 等能力可以拆模块接入。

5. 美颜美型截图：`assets/apk-demo/beauty-shape-controls.jpg`
   - 用途：说明商业美颜面板和五官精修入口。
   - 重点：展示美颜、美型、瘦脸、五官调整等常见产品模块。

## 公开展示边界

当前素材只包含 Demo App 操作截图与 GIF，不包含：

- 正式 SDK 二进制
- 模型权重
- 私有源码
- APK 安装包
- 商业授权密钥
- 客户定制资料

如需对外发送，建议只分享 GitHub 仓库链接或 README 页面，不单独发送工程目录。
